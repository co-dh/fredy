## Blocker analysis (verified 2026-06-16, two independent sub-agent probes + manual)

Both sorries are the genuine distributivity content of §1.616 (Freyd: "Direct images always
preserve unions").  They are NOT one-liners; each is blocked on infrastructure that does not yet
exist in the repo.  Neither weakening nor faking is acceptable, so the faithful `sorry`s remain.

### `compose_union_right` — the self-contained cover route is blocked on coproduct extensivity

I attempted the same cover⊥mono technique that proves `relUnion_le_reciprocal`.  Full
reconstruction (all steps below typecheck green; only the last fails):

1. `mST = case (pair S.colA S.colB)(pair T.colA T.colB)`; `image.lift mST : coprod S.src T.src →
   (S∪ᵣT).src` is a `Cover` (`image_lift_cover`).
2. `pbST = pullback(R.colB, (S∪ᵣT).colA)`; pull the cover `image.lift mST` back along `pbST.π₂`
   (needs `[PullbacksTransferCovers 𝒞]` via `cover_pullback`) → `Q` with cover leg `q₂ : Q → pbST.pt`
   and `q₁ : Q → coprod S.src T.src`.
3. Pull `q₁` back along `inl`/`inr` → `Q_S, Q_T` with legs `s₁ : Q_S → Q`, `t₁ : Q_T → Q`.
4. Build `dS : Q_S → U.src`, `dT : Q_T → U.src` (U = (R⊚S)∪ᵣ(R⊚T), pU = pair U.colA U.colB monic)
   with `dS ≫ pU = s₁ ≫ q₂ ≫ spanST` and the symmetric `dT`.  Assemble `cmp = case s₁ t₁` and
   `dJoin = case dS dT`; prove `dJoin ≫ pU = cmp ≫ (q₂ ≫ spanST)` by `case_uniq`.  ALL GREEN.
5. **BLOCKED:** to descend `dJoin` to a single `d : Q → U.src` via `cover_mono_diagonal` one needs
   `Cover cmp`, i.e. `Cover (case s₁ t₁)` — exactly that the two pullbacks of `q₁` along the
   coproduct injections JOINTLY COVER `Q`.  That is **coproduct extensivity** (universal + disjoint
   coproducts), strictly stronger than `RegularCategory`/`PullbacksTransferCovers`.

The repo has no extensivity / `Cover (case _ _)` lemma.  `disjoint_cover_is_coproduct`
(S1_61.lean:298) is the CONVERSE (disjoint cover ⟹ coproduct), not coproduct ⟹ joint cover.
`relUnion_le_reciprocal` escapes this because reciprocation is an iso (`pair snd fst`), so the
copairing transports directly with no splitting; composition is not an iso, so it does not.

### Both sorries — the book's route needs a BinRel↔Subobject(A×B) bridge that is missing

Freyd derives §1.616 from `PreLogos.invImage_preserves_union` via two identities:
  `yS = (y×1)#(S)` (precompose by a graph = inverse image along `y×1`) and
  `x∘(yS) = (x×1)(yS)` (postcompose = DIRECT image along `x×1`).
But `relUnion`/`intersect`/`compose` are defined at the **BinRel** level (image of a copairing /
pullback of pairings / image of a span in A×B), while `PreLogos.invImage_preserves_union` is stated
for the **abstract** `HasSubobjectUnions.union` field on `Subobject (prod A B)`.  These two unions
are never identified.  Missing pieces (confirmed absent by grep):
  - no `BinRel 𝒞 A B → Subobject 𝒞 (prod A B)` conversion (the monic `pair R.colA R.colB` is used
    ad hoc but never packaged), nor the inverse;
  - no theorem `relUnion R S ≅ HasSubobjectUnions.union ⟨pairR⟩ ⟨pairS⟩`;
  - no `yS = (y×1)#S` (only `graphMono_prodMap`, S1_92.lean:139, a fragment);
  - no `x∘(yS) = (x×1)(yS)` (composition-as-direct-image) at all.

### What would unblock (precise, for a follow-up)

Add to S1_56/S1_51 (NOT S1_60, which is import-frozen here): (a) `relSub : BinRel 𝒞 A B →
Subobject 𝒞 (prod A B)` and its lattice-iso to `relUnion`/`intersect`; (b) `compose_eq_directImage`:
`R ⊚ S = image((R.colB-pullback)≫(prodMap …))` recast as a direct image of `relSub S` along a
product map; (c) the elementary fact `directImage f (S ∪ T) = directImage f S ∪ directImage f T`
(image preserves joins — provable from `image_min` + `le_relUnion`, no extensivity).  Then
`compose_union_right` is (c) specialized, and `rel_inter_union_le` follows from
`invImage_preserves_union` through (a)+(b).  ALTERNATIVELY add coproduct extensivity and finish the
cover route in step 5 above (the rest is already proven to typecheck).

## Structure

- `HasSubobjectUnions` — existence of joins in `Sub(B)` for each `B`
- `InverseImage` — pullback of a subobject along a morphism
- `inverseImage_preserves_unions` — predicate: `f*(S₁ ∪ S₂) = f*S₁ ∪ f*S₂`
- `PreLogos` — extends `RegularCategory` + `HasSubobjectUnions` + `invImage_preserves_union`
- `IsDistributiveLattice` / `poset_prelogos_iff_distributive` — §1.616 BinRel(A,B) is distributive iff PreLogos
- `relUnion` — image of `case(pairR, pairS) : R.src+S.src → A×B`
