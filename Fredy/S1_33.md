## Resolved sorries (both discharged, zero axioms)

### `combined_reflects_leftInv` — DONE (statement corrected)
The old statement was unprovable *as written*: hypothesis `l ∘ combinedCayleyMap x = id`
is, in the repo's Cat-of-Types diagram-order convention, `HasRightInv (F x)`, and a careful
analysis shows that from a function *left* inverse of `F(x)` one extracts only that `x` is
monic (C component) plus `x ≫ z = id_A` (P(C°) component, surjectivity of `contraCayleyMap x`)
— i.e. `HasRightInv x`, never the wanted `y ≫ x = id_B` (`HasLeftInv x`).

To match the conclusion `y ≫ x = id_B` the hypothesis must be a function **right** inverse
`r` of `combinedCayleyMap x` (`combinedCayleyMap x ∘ r = id`); that makes `F(x)` surjective,
hence its covariant-Cayley component `covCayleyMap x` surjective, and surjectivity onto
`⟨B, id_B⟩ ∈ C(B)` yields `⟨W, y⟩` with `y ≫ x = id_B`.  Proof mirrors the already-proven
`contraCayley_reflects_rightInv` (generalize the preimage element, split the sigma, project).
This is the genuine book statement (Freyd: "F reflects both left and right invertibility");
the P(C°) half is the dual reflecting right-invertibility.

### `proset_faithful_iff_injective` — DONE (statement sharpened with fullness)
"Faithful ⟹ injective on objects" is FALSE with our `Faithful = Embedding + reflects-iso`
*without* fullness.  Counterexample: the unique functor from the 2-element discrete preorder
{a,b} to the 1-element preorder {x} is an Embedding (no non-trivial parallel morphisms) and
reflects isos (vacuously), yet is not injective on objects.  The book's §1.333 uses the
induced-ordering (full) representations, so the iff is stated under a `hFull` hypothesis,
which makes it genuinely true: forward = `proset_full_faithful_inj` (uses fullness + antisymP),
backward = inline (`proset_inj_faithful`'s argument), fullness unused.  `proset_full_faithful_inj`
was moved above this theorem so it can be reused for the forward direction.
