## Design note: `Logos` now carries the lattice bottom

Freyd §1.7 / §1.6 define a logos's subobject structure as a **lattice** "not just a
semi-lattice", i.e. each `Sub(A)` has *all* finite joins including the **empty join**
(the minimal subobject `0`, §1.61).  The old Lean `Logos` only carried `HasSubobjectUnions`
(binary joins), so it was strictly weaker than the book's logos and `logos_implies_preLogos`
could not supply the four `PreLogos` bottom fields.

Fix (book-faithful, in scope): `Logos` now adds the three bottom data/prop fields
(`bottom`, `bottom_min`, `bottom_dom_iso`) mirroring `PreLogos`'s lattice bottom.  Then:

- `bottom`, `bottom_min`, `bottom_dom_iso` of the resulting `PreLogos` are **passed through**.
- `invImage_preserves_bottom` is **derived** (`logos_invImage_pres_bottom`): §1.711 observes
  f# is a *left* adjoint (`f# ⊣ f##`), so it preserves the empty join. Concretely the
  adjunction turns `f#(⊥_B) ≤ A'` into `⊥_B ≤ rightAdj f A'` (true by `bottom_min`); taking
  `A' = ⊥_A` gives `f#(⊥_B) ≤ ⊥_A`, the reverse is `bottom_min`, and antisymmetry
  (`Subobject.le_antisymm_iso`) gives the required domain iso. Axiom-free.

The single in-file `Logos` constructor `locallyComplete_with_union_preserving_is_logos`
inherits the bottom from its `PreLogos` hypothesis `PL` (`PL.bottom`/`PL.bottom_min`/
`PL.bottom_dom_iso`), so no new proof obligation leaks out.  Downstream files
(`S1_72 FocalLogos`, `S1_77`) only `extends Logos` abstractly — no concrete constructor
breaks.

## §1.71 resolved: Boolean-algebra strengthening of `HasSubobjectComplements`

The old `HasSubobjectComplements` was an **under-specified definition**: its `compl_join`
field only asserted join *commutativity* `S ∪ ¬S = ¬S ∪ S` — a law that holds in *any*
lattice and carries no Boolean content.  It was dropped and the class is now the genuine
Boolean algebra on each `Sub(A)` (Freyd §1.71), parametrised by `[HasSubobjectUnions 𝒞]`:

- `compl_inv` — involution `¬¬S = S` (kept);
- `compl_antitone` — `¬` reverses the order `S ≤ T ⟹ ¬T ≤ ¬S`;
- `excluded_middle` — `S ∪ ¬S = ⊤`, as `(entire A).le (S ∪ ¬S)`;
- `contradiction` — `S ∩ ¬S = ⊥`, order-form: any common lower bound of `S` and `¬S` lies
  below `¬⊤` (the Boolean bottom = complement of top, so no separate `bottom` field needed).

`compl_le_compl_iff` (**complement contravariance** `X ≤ ¬Y ⟺ Y ≤ ¬X`) is then *derived*
from involution + antitonicity — this is the bridge Freyd's chain turns on.

### `boolean_logos_rightAdj_eq_compl_direct_compl` — fully proven, axiom-free

`lean_verify` ⇒ `{"axioms":[],"warnings":[]}` (no `sorryAx`, no choice).  The proof is
Freyd's verbatim order biconditional
`B' ⊆ ¬f(¬A') ⟺ f(¬A') ⊆ ¬B' ⟺ ¬A' ⊆ f#(¬B') ⟺ ¬f#(¬B') ⊆ A' ⟺ f#(B') ⊆ A'`,
combined with the right-adjoint adjunction `f#(B') ⊆ A' ⟺ B' ⊆ f##(A')`: both `¬f(¬A')` and
`f##(A')` are characterised by the same down-set `{B' | f#(B') ⊆ A'}`, hence each ≤ the other.

Two facts live genuinely *across* `f` (not inside a single `Sub(A)` Boolean algebra) and are
taken as **honest hypotheses of the theorem** — they are not fabricated or assumed globally,
the caller must supply them, and they are true in any boolean pre-logos:

- `directImage_adj` — the **direct-image ⊣ inverse-image** Galois connection
  `f(X) ⊆ Y ⟺ X ⊆ f#(Y)` (standard regular-category image/pullback fact, not yet developed
  as a standalone lemma in this repo);
- `invImage_compl` — **f# preserves complements** `¬f#(¬B') = f#(B')`, exactly the identity
  Freyd writes "now use that f# preserves complements".

Instance coherence: the `BooleanLogos` section now draws `HasImages`/`HasPullbacks` solely
from the single `HasRightAdjointImage` instance (dropping the standalone `[HasPullbacks]`/
`[HasImages]` variables), so every `InverseImage`/`DirectImage`/`compl` shares one pullback
structure — the same two-instances pitfall the §1.711 helper sidesteps.
