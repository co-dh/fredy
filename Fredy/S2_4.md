## Resolved blocker: eps_thick axiom (was the §2.4 root blocker)

The previous axiom `eps_thick (b) : 1 ⊑ ∋/∋` was **vacuously true** in any division allegory
(`one_le_div_self`), carrying no information. It has been replaced by Freyd's own §2.413 form,
which IS the thickness condition:

```
eps_thick {b c} (R : c ⟶ b) : ∃ (f : c ⟶ powerObj b), Map f ∧ f ≫ eps b = R
```

"for every R targeted at b there exists a map f with f∋ = R." §2.413 proves this forces
`A(R) = R/ₛ∋` entire (and §2.41's third containment `1 ⊑ A(R)A°(R)` is exactly this entireness).
This is faithful to the book (it is the condition Freyd uses to *infer* thickness) and is
constructive + non-vacuous. Changing it was in-scope (the class lives in S2_4) and broke no
downstream code (`eps_thick` had no other users; no `PowerAllegory` instances exist yet).

With this fix the whole §2.41–§2.421 chain goes through constructively (axioms: `propext` only):
- `A_is_map` entire branch: thickness witness `f ⊑ A R` (via `thick_witness_le_A`), so
  `1 ⊑ ff° ⊑ (A R)(A R)°` ⟹ `dom(A R) = 1`.
- `A_eps_eq` lower bound: `R = f∋ ⊑ (A R)∋` since `f ⊑ A R`.
- `A_of_map`: `f≫A(1)` is a map (`map_comp`) with `(f≫A(1))∋ = f` (`A_eps_eq`), so `=A(f)` by `A_unique`.
- `symm_div_eq_A_comp` lower bound: `(R/ₛS)°≫A R ⊑ A S` (symmDiv char), reciprocate, then use
  `A R` entire to insert `A R ≫ (A R)°` in front of `R/ₛS`.
