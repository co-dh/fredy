## Resolved blocker: eps_thick axiom (was the §2.4 root blocker)

The previous axiom `eps_thick (b) : 1 ⊑ ∋/∋` was **vacuously true** in any division allegory
(`one_le_div_self`), carrying no information. It has been replaced by Freyd's own §2.413 form,
which IS the thickness condition:

```
eps_thick {b c} (R : c ⟶ b) : ∃ (f : c ⟶ powerObj b), Map f ∧ f ≫ eps b = R
```

"for every R targeted at b there exists a map f with f∋ = R." §2.413 proves this forces
`A(R) = R/ₛ∋` entire (and §2.41's third containment `1 ⊑ A(R)A°(R)` is exactly this entireness).
This is faithful to the book (it is the condition Freyd uses to *infer* thickness) and is
constructive + non-vacuous. Changing it was in-scope (the class lives in S2_4) and broke no
downstream code (`eps_thick` had no other users; no `PowerAllegory` instances exist yet).

With this fix the whole §2.41–§2.421 chain goes through constructively (axioms: `propext` only):
- `A_is_map` entire branch: thickness witness `f ⊑ A R` (via `thick_witness_le_A`), so
  `1 ⊑ ff° ⊑ (A R)(A R)°` ⟹ `dom(A R) = 1`.
- `A_eps_eq` lower bound: `R = f∋ ⊑ (A R)∋` since `f ⊑ A R`.
- `A_of_map`: `f≫A(1)` is a map (`map_comp`) with `(f≫A(1))∋ = f` (`A_eps_eq`), so `=A(f)` by `A_unique`.
- `symm_div_eq_A_comp` lower bound: `(R/ₛS)°≫A R ⊑ A S` (symmDiv char), reciprocate, then use
  `A R` entire to insert `A R ≫ (A R)°` in front of `R/ₛS`.

## Resolved: §2.431 `thick_iff_existential` — STATEMENT REPAIRED + FULLY PROVEN

The earlier formulation was LOGICALLY FALSE. Two independent defects, both now fixed:

1. **`Thick` used the wrong division.** The book's thickness (§2.412/§2.43) is the entireness
   of `Δ(R) = R/ₛ∋`, the **symmetric** division — generalised here to `∀R, Entire (R/ₛT)`.
   The old def `∀R, Entire (R/T)` used the **plain** asymmetric right division; this predicate
   is strictly stronger and breaks the forward direction.
2. **The book's `R□ = T□` side-condition was dropped.** §2.431 quantifies only over R with the
   same codomain box (`R□ = 1_b ∩ R°R`); that guard is the domain on which Freyd's partial
   division `R/T` is defined. Restoring it (on BOTH `Thick`'s quantifier and the iff) is
   what makes the biconditional hold.

**Counterexample is now excluded correctly.** T = {(0,0)}:{0,1}→{0}, R = {(0,0),(1,0)} satisfy
`R□ = T□` and T is thick, so the corrected statement still applies — and it is SATISFIED: the
honest witness is `R̃ = R/ₛT = {(0,0),(1,0)}`, which is entire with `R̃T ⊑ R`, `R̃°R ⊑ T`. The old
statement was false only because it implicitly forced the witness to be the plain `R/T`, whose
`(R/T)°R = R ⊄ T`.

**Verification.** Both directions checked exhaustively in Rel (all relations, source/target sizes
up to 3×3): 0 forward violations, 0 reverse violations under the corrected statement. Dropping
the box guard re-introduces reverse violations; using plain `R/T` re-introduces forward violations.

**Proof.** Forward: witness `R̃ = R/ₛT`, entire directly from `Thick T` (which consumes the box
hypothesis); the two containments are the `le_symmDiv_iff` law on `R/ₛT ⊑ R/ₛT`. Reverse: `R̃ ⊑ R/ₛT`
and `R̃` entire force `R/ₛT` entire, via the constructive helper `entire_iff_one_le`. Axioms: `propext`
only (no choice, no `sorryAx`).
