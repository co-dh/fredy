# §2.435–§2.436 — The diagonal proofs (Cantor), algebraic form

Freyd & Scedrov, *Categories and Allegories*, Chapter 2, §2.4 (Power allegories), the "diagonal proofs"
subsection.  Formalized in `Fredy/S2_43.lean`.

## What the book says

- **§2.435.** Cantor: for any set `A`, its power-object `[A]` cannot appear as a sub-quotient of `A`; the same
  holds in any non-degenerate topos.  Freyd converts this to an algebraic statement: if a power allegory has a
  morphism `F : A → [A]` with `F°F = 1` (a partial map covering `[A]`), then `T = F∋` is thick, so `A` would
  carry a thick endomorphism.  But: *if a connected division allegory has a thick endomorphism then it is
  equivalent to the one-object one-morphism allegory.*  This reduces to:

- **§2.436.** *The equational theory of one-object pre-power allegories is inconsistent.*
  Define the "diagonal" relation `R = 𝟘 / (1 ∩ T)`, characterised by `S ⊑ R  ↔  S(1∩T) = 𝟘`.  By §2.431 there is
  a witness `R̄` (`= R/T`) with

  - (i)  `1 ⊑ R̄ R̄°`         (entire),
  - (ii) `R̄ T ⊑ R̄`,
  - (iii)`R̄° R̄ ⊑ T`,         and "hence `R̄ ⊑ R`".

  Freyd's diagonal chain is then

  ```
  R̄  ⊑  R̄ (1 ∩ R̄° R̄)  ⊑  R̄ (1 ∩ T)  ⊑  R (1 ∩ T)  ⊑  𝟘 ,
  ```

  so `R̄ = 𝟘`, and entireness (i) forces `1 ⊑ R̄ R̄° = 𝟘`, i.e. `1 = 𝟘`.

## The box guard is a HYPOTHESIS, not a sorry (the honest resolution)

The earlier draft carried `one_object_pre_power_inconsistent : Thick T → 1 = 𝟘` with a
`sorry` discharging an internal `have hBox : codBox (diag T) = codBox T`.  That `have`
is a **false statement**, and so is the unconditional theorem itself — both verified, so
the `sorry` was illegitimate (a sorry inside a false statement).  The fix restores
Freyd's own suppressed side-condition.

`one_object_pre_power_inconsistent` invokes `thick_iff_existential` (§2.431, the faithful
**box-guarded** biconditional from `S2_4`) on `R = diag T`.  §2.431 only supplies a witness
for an `R` with `codBox R = codBox T`; Freyd's BECAUSE writes "as insured by [2.431]" and
silently assumes this.  So the theorem takes

    (hBox : codBox (diag T) = codBox T)

as an explicit hypothesis.  Under it the witness `W = diag T /ₛ T` satisfies `Entire W`,
`W ≫ T ⊑ diag T`, `W° ≫ diag T ⊑ T`, and the rest is the complete §2.436 chain:

- From `W ≫ T ⊑ diag T` + coreflexive idempotency of `1 ∩ T`:
  `(W ≫ (1∩T)) ≫ (1∩T) = W ≫ (1∩T) ⊑ diag T ≫ (1∩T) ⊑ 𝟘`, so `W ≫ (1∩T) = 𝟘`, giving
  `W ⊑ diag T` by `le_diag_iff`.
- Then `W° ≫ W ⊑ W° ≫ diag T ⊑ T`, and `inconsistency_core` closes the goal.

### Why `hBox` cannot be a lemma (verified, not asserted)

`box_guard_fails_at_id_unless_degenerate` **proves in Lean** that for `T = 1_a`
(`1 ∩ T = 1`):

    diag 1 = 𝟘 / 1 = 𝟘,   codBox (diag 1) = codBox 𝟘 = 1 ∩ 𝟘°≫𝟘 = 𝟘,
    codBox 1 = dom 1 = 1 ∩ 1≫1 = 1.

So `codBox (diag T) = 𝟘 ≠ 1 = codBox T` in any non-degenerate allegory — the box guard is
*false*, not merely unproven.  Worse, the unconditional `Thick T → 1 = 𝟘` is itself **false**
for the repo's box-guarded `Thick`: exhaustive brute force in `Rel` (all source sizes
`c ≤ 8`, plus a closed all-`c` argument) exhibits genuinely thick endomorphisms on a
2-element object with `1 ≠ 𝟘` — e.g. `T = {(1,0)} : {0,1}→{0,1}`, codBox `{(0,0)}`.  The
box guard added in §2.43 to make §2.431 a true *biconditional* is exactly what makes the
`diag T` box guard unattainable; the two cannot both hold for a reflexive `T` (where the
diagonal argument needs to run).  This is a genuine definitional tension Freyd's prose
elides, surfaced here by the formalization rather than hidden behind a sorry.
