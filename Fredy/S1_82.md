## Discharged this session

- **`(⇒)` of both adjoint-functor-theorem iffs** (GAFT §1.83, MGAFT §1.831): peeled off the
  EASY half of each iff as three standalone, axiom-free helpers and wired the `(⇒)` branch
  of each iff through them. Only the deep `(⇐)` solution-set/representability construction
  remains sorry'd in each iff.
  - **`preAdjointFunctor_of_adjunction`**: a left adjoint is a pre-adjoint functor. The unit
    `η_A : A → G(F A)` is itself a *singleton* (`I := PUnit`) pre-adjoint family: any
    `f : A → G B` factors as `η_A ≫ G(ψ f) = φ(ψ f) = f` (via `φ_eq` + `φψ`, §1.817).
  - **`isContinuous_of_adjunction`**: a right adjoint preserves all small limits. Transpose
    the W-cone `{W → G(D i)}` to `{F W → D i}` via `ψ` (cone-naturality from `ψ_nat_right`),
    lift through the limit to `w : F W → lim.apex`, transpose back `u := φ w`. Uniqueness:
    any `u'` gives `ψ u'` mediating ⟹ `ψ u' = w` by `lim.uniq` ⟹ `u' = φ(ψ u') = φ w`.
    `lean_verify`: axiom-free.
  - **`isUniformlyContinuous_of_adjunction`**: same transpose, but no completeness needed —
    take `B := F A`, `cone_legs i := ψ(legs i)`, factoring map `η_A`. Axiom-free.

- **`complete_hasEqualizers`** (§1.825 ⇒ equalizer side): built `WPPv := ULift WPP : Type v`
  with `wppCatV : Cat.{v}` (ULifted homs), the parallel-pair diagram functor `wppDiagObj/Map`,
  and read off the equalizer from `Complete.hasLimit`. The `src`-leg is `eqMap`, the lift comes
  from the limit lift on the cone `{botL ↦ c.map, …}`, and uniq from `lim.uniq`. This closes the
  `(⇒)` of `complete_iff_eq_prod`, so that whole iff is now proved.

- **`preserves_weaklim_iff_preserves_lim`** (§1.829, the uniqueness gap): faithful formalization
  of Freyd's actual argument "preservation of weak-limits ⟹ preservation of monic families".
  Introduced the book's J-poset as `MFShape 𝒟` (discrete `𝒟` + two bottom points `botL/botR`,
  each below every `pt i`), with `mfDiagObj`/`mfDiagMap` sending `pt i ↦ Dᵢ`, `botL/botR ↦ L`,
  `arrL i / arrR i ↦ πᵢ`. The canonical lower bound is a weak-limit **iff** `{πᵢ}` is collectively
  monic (`limMonic`, proved from `lim.uniq`). Feeding this through the weak-limit-preservation
  hypothesis yields a lift `w` with `w ≫ T(id_L) = u` and `w ≫ T(id_L) = u'`, hence `u = u'`.
  Verified axiom-free (`lean_verify` returns no axioms).
