## `pullback_faithful_iff_preserves_properness` (⇐) — DISCHARGED

The last sorry is gone.  The honest fix (Freyd §1.472, "representation of a Cartesian category") was to
add `PreservesProductMonic T` — the projections `(T fst, T snd)` stay a *monic pair* — to the
hypotheses, since `PreservesPullbacks` alone cannot force `T⟨1,f⟩ = T⟨1,g⟩` from `T f = T g`.
(Named distinctly from §1.437's `PreservesBinaryProducts`, which asks the comparison
`T(A×B) → TA×TB` to be iso and so needs `HasBinaryProducts ℬ` — unavailable here.)

Final proof (constructive modulo the permitted classical `by_cases`):

- **Reflecting isos** (`IsIso (T f) → IsIso f`): `T f` iso ⇒ mono ⇒ `f` mono (`reflectsMono`); if `f`
  were a proper mono, `PreservesProperness` would make `T f` non-iso — contradiction.  So `f` iso.
- **Embedding** (`T f = T g → f = g`):
  1. Equalizer of `f, g` = pullback `P` of `u = ⟨1,f⟩, v = ⟨1,g⟩ : A → A×B`.  Its two projections
     coincide (post-`fst`); `e := π₁` equalizes `f, g` and is monic.
  2. `T f = T g` + `PreservesProductMonic` ⇒ `T u = T v` (post `T fst` / `T snd`).
  3. `PreservesPullbacks` ⇒ image of `P` is a pullback; with `T u = T v` it is the kernel pair of the
     split-mono (hence monic) `T u`.  By `mono_iff_level_diag_iso` in `ℬ`, the diagonal `δ` is iso, and
     `δ ≫ T e = 1` then forces `T e` iso.
  4. `e` monic + `T e` iso ⇒ (properness contrapositive) `e` iso ⇒ cancel `e` in `e≫f = e≫g` ⇒ `f = g`.

Axioms: `propext`, `Classical.choice` (the `by_cases`), `Quot.sound` — no `sorryAx`.

### What remains (single faithful `sorry`, two goals), and the GENUINE blocker

`Faithful T = Embedding T ∧ (∀ f, IsIso (T f) → IsIso f)`.

1. **Embedding** (`T f = T g → f = g`).  Classical route with `reflectsMono` in hand: form the
   equalizer `e : E ↣ A` of `f, g` (exists — `𝒜` has products + pullbacks, §1.434).  If `f ≠ g`, `e`
   is a non-iso mono (proper), so `T e` is non-iso by `PreservesProperness`.  To contradict, need
   `T e` ISO when `T f = T g`, i.e. `T e` split by `1_{T A}`.
2. **Reflecting isos** (`IsIso (T f) → IsIso f`).  `IsIso (T f) ⇒ Mono (T f) ⇒ Mono f` is discharged
   inline by `reflectsMono`; the leftover `Mono f ⇒ IsIso f` needs the same splitting.

**Blocker (mathematical, not an axiom gap):** both need `T` to carry the equalizer-pullback to a
pullback whose apex is split by `1_{T A}`, which holds iff `T u = T v` for `u = ⟨1,f⟩, v = ⟨1,g⟩`.
That requires `T` to preserve the **binary product** `A × B` (equivalently, preserve equalizers):
`(T fst, T snd)` is not jointly monic under `PreservesPullbacks` alone.  Freyd resolves this in
**§1.472** by strengthening the hypothesis from "preserves pullbacks" to "**representation of
Cartesian categories**" (preserves the whole Cartesian structure).  So the honest fix is to add a
`PreservesBinaryProducts T` (or `PreservesEqualizers T`) hypothesis to the theorem — the statement as
currently written, with only `PreservesPullbacks`, is strictly under-hypothesised for the `Faithful`
direction in this repo's `Faithful` (which also bundles reflects-isos).
