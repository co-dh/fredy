## §2.352  Straight morphisms cancel

| Item | Lean decl | Status |

`straight_cancel_simple` (§2.352) is now fully proven. The key was the helper `le_dom_comp`:
`R ⊑ (dom R) ≫ R` holds for *every* R (not just entire ones) — it is the modular law
`modular_le 1 R R` after `Cat.id_comp` and `inter_idem`. (The previous note here was wrong:
it claimed this step needs entireness; it does not.) The proof then follows the book exactly:
from `FS=GS` and S straight, `G°F ⊑ S/ₛS ⊑ 1` and `F°G ⊑ 1`; then the chain
`(dom F)G ⊑ (dom F)(dom G)G = (dom G)(dom F)G ⊑ (dom G)F·F°G ⊑ (dom G)F`, using
coreflexive commutation `(dom F)(dom G) = (dom G)(dom F)` (`coreflexive_comp_eq_inter` +
`inter_comm`) and `dom F ⊑ FF°`. The reverse inclusion is symmetric, giving equality.

Note: this file's `⊑` has no registered `Trans`/`@[trans]` instance, so calc cannot chain two
`⊑` steps (only `=`→`⊑` and `⊑`→`=` work, via builtin `Eq` Trans). Inequality chains must use
explicit `le_trans`, not calc.

## §2.353  Converse of straight cancellation

| Item | Lean decl | Status |
| ---- | --------- | ------ |
| FS=GS for all simple F,G → (dom F)G=(dom G)F implies S straight | `straight_of_cancel` | SORRY |

FAITHFUL SORRY — sharpened blocker. The statement (cancellation-on-simples ⟹ `Straight S`)
matches §2.353 exactly. What is missing is a hypothesis on the ambient allegory, NOT the inner
algebra:

- **Reduction step (the actual blocker):** to prove `S/ₛS ⊑ 1` the book reduces to proving
  `F°G ⊑ 1` for every *simple* F, G with `F°G ⊑ S/ₛS`. This reduction is valid only because
  §2.225 ("every morphism is the union of the semisimple morphisms it contains") expresses
  `S/ₛS` as that union. §2.225 needs unions / local completeness, which the bare
  `DivisionAllegory` interface in this file does not provide, and §2.225 itself is unformalized.
  Note: even assuming `[SemiSimpleAllegory 𝒜]` (every morphism semisimple, `R = P ≫ Q°` with
  P, Q simple) does NOT directly give the generator form `F° ≫ G` with both F, G simple
  (`P ≫ Q° = (P°)° ≫ Q°` needs P°, Q° simple, which is not implied), so the one-line reduction
  is unavailable even there. A faithful proof needs §2.225's union-of-semisimple decomposition.

- **Inner argument (NOT the blocker; pure algebra, mechanizable):** given simple F, G with
  `F°G ⊑ S/ₛS`, set `F' = (dom G)F`, `G' = (dom F)G`. One shows `F'S = G'S`, so by the
  hypothesis `h` (applied to simple F', G') `dom F' ≫ G' = dom G' ≫ F'`, which forces `F' = G'`
  and hence `F°G = F'°G' ⊑ 1`. The `F'S ⊑ G'S` / `G'S ⊑ F'S` chains use only
  `(S/ₛS)S ⊑ S` and dom-algebra — the same toolkit that proved `straight_cancel_simple`.

To discharge: formalize §2.225 (union-of-semisimple-below) for a locally-complete distributive
division allegory, then the reduction + inner argument above close §2.353.
