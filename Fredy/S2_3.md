`straight_cancel_simple` (¬ß2.352) is now fully proven. The key was the helper `le_dom_comp`:
`R ‚äĎ (dom R) ‚Čę R` holds for *every* R (not just entire ones) ‚ÄĒ it is the modular law
`modular_le 1 R R` after `Cat.id_comp` and `inter_idem`. (The previous note here was wrong:
it claimed this step needs entireness; it does not.) The proof then follows the book exactly:
from `FS=GS` and S straight, `G¬įF ‚äĎ S/‚āõS ‚äĎ 1` and `F¬įG ‚äĎ 1`; then the chain
`(dom F)G ‚äĎ (dom F)(dom G)G = (dom G)(dom F)G ‚äĎ (dom G)F¬∑F¬įG ‚äĎ (dom G)F`, using
coreflexive commutation `(dom F)(dom G) = (dom G)(dom F)` (`coreflexive_comp_eq_inter` +
`inter_comm`) and `dom F ‚äĎ FF¬į`. The reverse inclusion is symmetric, giving equality.

Note: this file's `‚äĎ` has no registered `Trans`/`@[trans]` instance, so calc cannot chain two
`‚äĎ` steps (only `=`‚Üí`‚äĎ` and `‚äĎ`‚Üí`=` work, via builtin `Eq` Trans). Inequality chains must use
explicit `le_trans`, not calc.

## ¬ß2.353  Converse of straight cancellation

| Item | Lean decl | Status |
| ---- | --------- | ------ |
| FS=GS for all simple F,G ‚Üí (dom F)G=(dom G)F implies S straight | `straight_of_cancel` | **DONE** |
| ¬ß2.225 union-of-semisimple property (the book's stated hypothesis) | `UnionOfSemiSimple` | DONE (def) |

**DONE ‚ÄĒ sorry-free, axiom-free** (`lean_verify`: `axioms: []`, fully constructive).

### Statement is faithful; prior "type-level obstruction" was a misdiagnosis

The hypothesis `h` is stated exactly as Freyd writes it: `F G : c ‚ü∂ a` simple, same source `c`
and target `a`, `F ‚Čę S = G ‚Čę S ‚Üí dom F ‚Čę G = dom G ‚Čę F`. There is **no** source/target swap.
(The earlier note claimed `SemiSimple (S/‚āõS)` forces `F G : a ‚ü∂ c`; in fact the ¬ß2.353
reduction quantifies over book-semisimple pieces `F¬į ‚Čę G` with `F G : c ‚ü∂ a`, which compose
with `S : a ‚ü∂ b` correctly. The book's ¬ß2.16(10) defines semisimple as **`F¬įG`** with F, G
simple. `UnionOfSemiSimple` here quantifies over `F¬į ‚Čę G` directly, matching both the book and
the (now corrected) `SemiSimple` in S2_1, which also encodes `F¬į ‚Čę G`. (The S2_1 `SemiSimple`
formerly encoded the wrong recip `F ‚Čę G¬į`; that integrity bug has since been fixed.))

### What was genuinely required: the ¬ß2.225 hypothesis (and only that)

Freyd states ¬ß2.353 only "for division allegories in which every morphism is the union of the
semisimple morphisms it contains [2.225]". That union/local-completeness property is **not**
part of the bare `DivisionAllegory` interface and is unformalized for abstract allegories
(the repo's union infra lives only on the Subobject/relations side, S1_60/S1_72). So it is
added as an explicit hypothesis `hUnion : UnionOfSemiSimple (S /‚āõ S)`, encoded by its universal
property (`R` is the least upper bound of the semisimple pieces `F¬į ‚Čę G ‚äĎ R`). This matches the
book's stated condition exactly ‚ÄĒ the theorem is faithful, not weakened ad hoc.

### The inner argument (pure division-allegory algebra, fully discharged)

By ¬ß2.225 it suffices to show `F¬į ‚Čę G ‚äĎ 1` for all simple F, G with `F¬į ‚Čę G ‚äĎ S/‚āõS`. Set
`F' = (dom G) ‚Čę F`, `G' = (dom F) ‚Čę G`. Then:

- **`dom F' = dom G'`** ‚ÄĒ an *equality* (not the `dom G' ‚äĎ dom F'` the old note guessed). Both
  equal `dom F ‚ą© dom G`, via the new lemma `dom_coref_comp : dom (E ‚Čę F) = E ‚ą© dom F` for
  coreflexive E (proved from the coreflexive-sandwich identity `coref_sandwich`).
- **`F'¬į ‚Čę G' = F¬į ‚Čę G`** ‚ÄĒ `recip_comp_dom` + `dom_comm` + `dom_comp_self`.
- **`F' ‚Čę S = G' ‚Čę S`** ‚ÄĒ the book chain `G'S ‚äĎ F'F'¬įG'S ‚äĎ F'(S/‚āõS)S ‚äĎ F'S` and symmetric,
  using `(S/‚āõS)S ‚äĎ S` and `dom F' = dom G'`.
- Then `h F' G'` gives `dom F' ‚Čę G' = dom G' ‚Čę F'`; with `dom F' = dom G'` and
  `dom F' ‚Čę F' = F'` (`dom_comp_self`) this forces **`F' = G'`**, whence
  `F¬į ‚Čę G = F'¬į ‚Čę G' = F'¬į ‚Čę F' ‚äĎ 1` by simplicity of `F'`.
