`straight_cancel_simple` (§2.352) is now fully proven. The key was the helper `le_dom_comp`:
`R ⊑ (dom R) ≫ R` holds for *every* R (not just entire ones) — it is the modular law
`modular_le 1 R R` after `Cat.id_comp` and `inter_idem`. (The previous note here was wrong:
it claimed this step needs entireness; it does not.) The proof then follows the book exactly:
from `FS=GS` and S straight, `G°F ⊑ S/ₛS ⊑ 1` and `F°G ⊑ 1`; then the chain
`(dom F)G ⊑ (dom F)(dom G)G = (dom G)(dom F)G ⊑ (dom G)F·F°G ⊑ (dom G)F`, using
coreflexive commutation `(dom F)(dom G) = (dom G)(dom F)` (`coreflexive_comp_eq_inter` +
`inter_comm`) and `dom F ⊑ FF°`. The reverse inclusion is symmetric, giving equality.

Note: this file's `⊑` has no registered `Trans`/`@[trans]` instance, so calc cannot chain two
`⊑` steps (only `=`→`⊑` and `⊑`→`=` work, via builtin `Eq` Trans). Inequality chains must use
explicit `le_trans`, not calc.

## §2.353  Converse of straight cancellation

| Item | Lean decl | Status |
| ---- | --------- | ------ |
| FS=GS for all simple F,G → (dom F)G=(dom G)F implies S straight | `straight_of_cancel` | **DONE** |
| §2.225 union-of-semisimple property (the book's stated hypothesis) | `UnionOfSemiSimple` | DONE (def) |

**DONE — sorry-free, axiom-free** (`lean_verify`: `axioms: []`, fully constructive).

### Statement is faithful; prior "type-level obstruction" was a misdiagnosis

The hypothesis `h` is stated exactly as Freyd writes it: `F G : c ⟶ a` simple, same source `c`
and target `a`, `F ≫ S = G ≫ S → dom F ≫ G = dom G ≫ F`. There is **no** source/target swap.
(The earlier note claimed `SemiSimple (S/ₛS)` forces `F G : a ⟶ c`; in fact the §2.353
reduction quantifies over book-semisimple pieces `F° ≫ G` with `F G : c ⟶ a`, which compose
with `S : a ⟶ b` correctly. The book's §2.16(10) defines semisimple as **`F°G`** with F, G
simple — note the *repo's* `SemiSimple` in S2_1 encodes it as `F ≫ G°`, the wrong recip; to
stay book-faithful, `UnionOfSemiSimple` here quantifies over `F° ≫ G` directly.)

### What was genuinely required: the §2.225 hypothesis (and only that)

Freyd states §2.353 only "for division allegories in which every morphism is the union of the
semisimple morphisms it contains [2.225]". That union/local-completeness property is **not**
part of the bare `DivisionAllegory` interface and is unformalized for abstract allegories
(the repo's union infra lives only on the Subobject/relations side, S1_60/S1_72). So it is
added as an explicit hypothesis `hUnion : UnionOfSemiSimple (S /ₛ S)`, encoded by its universal
property (`R` is the least upper bound of the semisimple pieces `F° ≫ G ⊑ R`). This matches the
book's stated condition exactly — the theorem is faithful, not weakened ad hoc.

### The inner argument (pure division-allegory algebra, fully discharged)

By §2.225 it suffices to show `F° ≫ G ⊑ 1` for all simple F, G with `F° ≫ G ⊑ S/ₛS`. Set
`F' = (dom G) ≫ F`, `G' = (dom F) ≫ G`. Then:

- **`dom F' = dom G'`** — an *equality* (not the `dom G' ⊑ dom F'` the old note guessed). Both
  equal `dom F ∩ dom G`, via the new lemma `dom_coref_comp : dom (E ≫ F) = E ∩ dom F` for
  coreflexive E (proved from the coreflexive-sandwich identity `coref_sandwich`).
- **`F'° ≫ G' = F° ≫ G`** — `recip_comp_dom` + `dom_comm` + `dom_comp_self`.
- **`F' ≫ S = G' ≫ S`** — the book chain `G'S ⊑ F'F'°G'S ⊑ F'(S/ₛS)S ⊑ F'S` and symmetric,
  using `(S/ₛS)S ⊑ S` and `dom F' = dom G'`.
- Then `h F' G'` gives `dom F' ≫ G' = dom G' ≫ F'`; with `dom F' = dom G'` and
  `dom F' ≫ F' = F'` (`dom_comp_self`) this forces **`F' = G'`**, whence
  `F° ≫ G = F'° ≫ G' = F'° ≫ F' ⊑ 1` by simplicity of `F'`.
