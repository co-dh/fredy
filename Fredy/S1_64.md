## Key Blockers

### `amalgamation_lemma` (В§1.651) and `pretopos_balanced` (В§1.652)
Both require **disjointness of coproducts**: `inl(b) вү  inr(c)` as a categorical axiom.
This is Freyd's *positivity* axiom (В§1.62): coproducts are disjoint and universal.
Currently `PositivePreLogos` only has the bare coproduct universal property via `HasBinaryCoproducts`;
it does NOT include disjointness or universality. Adding these axioms would unblock:
- `amalgamation_lemma`: the equivalence relation E on B+C requires disjointness for transitivity
- `pretopos_balanced`: the cokernel-pair of a monic lives in B+B; disjointness forces monicity

Fix: add to `PositivePreLogos` the field
  `coprod_disjoint : вҲҖ {A B : рқ’һ} (f : A вҹ¶ HasBinaryCoproducts.coprod A A), ...`
or use a dedicated `DisjointCoproducts` typeclass (disjointness + universality of coproducts).

### Session log вҖ” coproduct-disjointness pass (this session)

**Question answered: is coproduct disjointness a book DEFINITION or a DERIVED theorem?**
вҶ’ It is **DEFINITIONAL** (a defining property of a *positive* pre-logos), NOT derivable from
the bare `HasBinaryCoproducts` universal property.  The book is explicit:

- **В§1.623** "A POSITIVE PRE-LOGOS is a pre-logos in which for every pair of objects A, B there
  exists [a coproduct] вҖҰ We can choose C so that A вҲӘ B = C."  The coproduct is *constructed* as
  the ambient object in which A, B sit as subobjects with `A вҲӘ B = C`.
- **В§1.621** "If AвӮҒ, AвӮӮ are subobjects of A in a pre-logos such that `AвӮҒ вҲ© AвӮӮ = 0` and
  `AвӮҒ вҲӘ AвӮӮ = A` then A is a coproduct of AвӮҒ, AвӮӮ."  So a positive coproduct IS a disjoint
  (`вҲ© = 0`), complemented (`вҲӘ = whole`) union вҖ” disjointness is part of the data.
- **В§1.626** "Coproducts can exist without positivity.  Any distributive lattice вҖҰ is a pre-logos
