## Discharged this pass

- **`membershipMap_nameOf`** (new, §1.942 β-law).  The membership test of the name
  `'A''` reconstructs the classifier:  `membershipMap (nameOf m hm) = classify m hm`.
  Proof: `nameOf m hm = curry(fst ≫ χ_m)`; factor `pair id (term≫N) = pair id term ≫
  prodMap … N`; `curry_eval_eq` collapses `prodMap … N ≫ eval = fst ≫ χ_m`; then
  `fst (pair id term) = id`.  This is Freyd's "(A → 'A'' → Ω) = χ_{A'}" step (§1.942).

- **`inter_le_named`** (§1.943 pt1).  Using `membershipMap_nameOf`, `membershipMap
  F_name = χ_{A'}`.  `interIntersection` is the canonical pullback of `true` along
  `membershipMap F_name`; `classify_pullback` makes `A'` a pullback of `true` along
  `χ_{A'}`; its universal property lifts the canonical cone to `u : ∩F.dom → A'.dom`
  with `u ≫ A'.arr = ∩F.arr`, i.e. `∩F ≤ A'`.

- **`singletonMap`** (§1.94(10)).  Was a `sorry` definition; now defined by reuse of
  the §1.92 `singletonMapCat A = curry(χ_{Δ_A})` (DRY).  Same object, same type
  (`powObj A = exp A Ω`).

- **`singletonMap_monic`** (§1.94(10), replaces the false `singletonMap_entire`).
  The old `singletonMap_entire : Cover (singletonMap A)` was NOT Freyd's statement
  and is mathematically false (A1 is monic, iso only for subterminal A).  Replaced
  with the genuine categorical fact — A1 is MONIC (§1.92) — via `singletonMapCat_monic`.

## Remaining sorries (faithful statements, genuinely deeper)

- **`inter_is_glb`** (§1.943 pt2): needs the *converse* of `inter_le_named` for a
  single point plus that `Ω(−)` (contravariant, with right adjoint) carries the
  well-pointed jointly-epic family `{1_F}` to a jointly-monic family.  That functor
  and adjunction are not yet available.

- **`topos_has_strict_coterminator`** (§1.944): `inter_is_glb` at `A=1, F=∅` gives the
  minimal subobject of 1; the strict-coterminator criterion (§1.58/§1.331) then closes.

- **`topos_is_regular`** (§1.945): image of `f` = `∩{B' | f factors through B'}` via
  `inter_is_glb` + the capitalization lemma (§1.54); plus `PullbacksTransferCovers`.

- **`topos_is_logos`** (§1.946): binary `A₁ ∪ A₂ = ∩{B' | A₁⊆B' ∧ A₂⊆B'}` and the
  right adjoint `f##` from the exponential adjunction; depends on §1.943 + §1.945.

- **`topos_has_rtc`** (§1.947): `inter_is_glb` for the family `F₁ ∩ F₂` of reflexive
  pre-closed relations on `B×B`; then ordinary relational reasoning.

- **`capital_is_solvable`** (§1.94(10)): capitalization lemma §1.54 + `inter_is_glb` +
  strict coterminator (for the `A* = 0` branch).

## MISSING (cannot be stated faithfully with current infrastructure)

- **§1.941 permanent lower bound under representations**: "for every representation
  of topoi `T : 𝒞 → 𝒟`, `T(∩F)` is below every subobject named by `T(F)`, and `∩F`
  is the largest subobject with this property."  Requires functorial transport of
  subobjects and of the `NamedBy` relation along `T` (a `T(F)`-as-subobject
  construction).  The bare `ToposMap` predicate carries none of this.  The
  *non-representational* fragment ("∩F ≤ every named subobject") IS proved as
  `inter_le_named`.  The old `inter_permanent_lower_bound : True := trivial` was a
  vacuous placeholder and has been **removed** (replaced by an explanatory comment).

- **§1.94(10) "U(A1) is always entire"**: entireness of the singleton map's image
  under the forgetful representation `U : 𝒞 → Set`.  Requires `U`, which this repo
  does not have.  (Note `Cover (singletonMap A)` in 𝒞 is *false*, so the previous
  `singletonMap_entire` was not the book's statement.)

## §1.948 (remark, no formal statement)

G-sets example shows ∩F can be ∅ even when the family of named subobjects is
non-empty (the forgetful `U : YG → Y` witnesses this via `U(∩F) = ∩(UF) = ∅`).
