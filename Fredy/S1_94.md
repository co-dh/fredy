## Discharged this pass

- **`membershipMap_nameOf`** (new, §1.942 β-law).  The membership test of the name
  `'A''` reconstructs the classifier:  `membershipMap (nameOf m hm) = classify m hm`.
  Proof: `nameOf m hm = curry(fst ≫ χ_m)`; factor `pair id (term≫N) = pair id term ≫
  prodMap … N`; `curry_eval_eq` collapses `prodMap … N ≫ eval = fst ≫ χ_m`; then
  `fst (pair id term) = id`.  This is Freyd's "(A → 'A'' → Ω) = χ_{A'}" step (§1.942).

- **`inter_le_named`** (§1.943 pt1).  Using `membershipMap_nameOf`, `membershipMap
  F_name = χ_{A'}`.  `interIntersection` is the canonical pullback of `true` along
  `membershipMap F_name`; `classify_pullback` makes `A'` a pullback of `true` along
  `χ_{A'}`; its universal property lifts the canonical cone to `u : ∩F.dom → A'.dom`
  with `u ≫ A'.arr = ∩F.arr`, i.e. `∩F ≤ A'`.

- **`singletonMap`** (§1.94(10)).  Was a `sorry` definition; now defined by reuse of
  the §1.92 `singletonMapCat A = curry(χ_{Δ_A})` (DRY).  Same object, same type
  (`powObj A = exp A Ω`).

- **`singletonMap_monic`** (§1.94(10), replaces the false `singletonMap_entire`).
  The old `singletonMap_entire : Cover (singletonMap A)` was NOT Freyd's statement
  and is mathematically false (A1 is monic, iso only for subterminal A).  Replaced
  with the genuine categorical fact — A1 is MONIC (§1.92) — via `singletonMapCat_monic`.

## INTEGRITY FIX this pass — `inter_is_glb` → `inter_le_singleton_named`

Audit found the old `inter_is_glb` over-promised: its conclusion was the *per-singleton*
bound `∀ x, L ≤ interIntersection (x ≫ F.arr)`, NOT the glb `L ≤ ⋂F`, and its
`WellPointed F.dom` hypothesis (`_hF`) was unused.  The name claimed a glb the proof
never delivers — and the genuine glb cannot even be *stated* here, because
`interIntersection` only takes a single global name `1 → [A]` (a singleton family);
there is no `⋂F`-over-a-subobject-family construction in this file (that needs §1.54's
`capitalization_lemma`, still `sorry`).

Resolution: **renamed to `inter_le_singleton_named`** to say exactly what is proved —
for each point `x` of `F`, the singleton intersection `∩{x ≫ F.arr}` is itself NAMED
by F (its name is `x ≫ F.arr`, witnessed by `x`, via `nameOf_interIntersection`),
hence any `L` below every F-named subobject is below it.  The unused `WellPointed`
hypothesis was dropped (the per-singleton statement does not use it); `hL` is used.
The proof is unchanged and still sorry-free.  No name/statement now lies.

## §1.947 `topos_has_rtc` DISCHARGED this pass (via the new S1_77 keystone)

`Fredy/S1_77.lean` now provides the reflexive-transitive-closure assembly:
`transRefClos_of_glb_preclosed` turns the §1.947 family-glb shape `M = ⋂{S | 1⊑S ∧ R⊚S⊑S}`
(reflexive + R-stable + glb) plus a self-quotient `M/M` into a full `TransRefClos R`; and
`HasReflTransClosure` packages "every endo-relation has R*" — Freyd's own "a topos is a
transitive logos" hypothesis (§1.77).

**What changed.**  The old `topos_has_rtc` was a `sorry` stated in the *degenerate*
`Subobject`-encoding where "reflexive" was `Subobject.entire (B×B) ≤ R*` — i.e. the *entire*
B×B below R*, which forces R* = ⊤ (and its hypothesis `entire ≤ R` forced R = ⊤ too).  That
is not Freyd's reflexivity.  It is now **restated in the genuine §1.77 `BinRel` encoding**
(reflexivity = `graph(id) ⊑ R*`, the diagonal; transitivity = `R*⊚R* ⊑ R*`) and **discharged
sorry-free**:

```
theorem topos_has_rtc [HasImages 𝒞] [PullbacksTransferCovers 𝒞] [HasReflTransClosure 𝒞]
    {B : 𝒞} (R : BinRel 𝒞 B B) :
    ∃ Rstar, RelLe R Rstar ∧ IsReflexive Rstar ∧ IsTransitive Rstar ∧
      ∀ S, RelLe R S → IsReflexive S → IsTransitive S → RelLe Rstar S :=
  ⟨rtc R, le_rtc R, rtc_reflexive R, rtc_transitive R, fun S h₁ h₂ h₃ => rtc_minimal R S h₁ h₂ h₃⟩
```

**Honest split of "done" vs "residual".**  The *closure-ASSEMBLY* — R ⊑ R*, reflexive,
transitive, and minimality among reflexive-transitive relations ⊇ R — is now genuinely proved
(each of `le_rtc`/`rtc_reflexive`/`rtc_transitive`/`rtc_minimal` is `#print axioms`-clean in
S1_77).  The *residual* is **exactly the glb-existence**, carried faithfully by the
`[HasReflTransClosure 𝒞]` hypothesis: constructing that instance for a bare topos still needs
the §1.943 family-glb `M = ⋂F`, i.e. §1.54's `capitalization_lemma` (still `sorry`).  The two
regular-structure prerequisites `[HasImages 𝒞]`/`[PullbacksTransferCovers 𝒞]` (= the still-
`sorry` `topos_is_regular`, §1.945) are likewise carried as explicit hypotheses.  So the
theorem now isolates precisely the part S1_77 closes (assembly) from the part §1.543 owes
(glb-existence) — no false statement, no vacuity, hypotheses all used.

`#print axioms topos_has_rtc` still shows `sorryAx`, but only via the file-wide leak of
S1_92's `topos_has_exponentials` `sorry`-instance (synthesised from `[Topos 𝒞]`, which
`BinRel`'s `HasPullbacks` routes through) — the same leak that taints `inter_le_named`,
`membershipMap_nameOf`, and every other completed declaration here.  Not specific to this
discharge; S1_92 is off-limits.
