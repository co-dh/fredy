### `ac_iff_iac_and_projective_one` (§1.974) — AC ⇔ IAC ∧ 1 projective

Statement carries an extra `[PullbacksTransferCovers 𝒞]` (the topos-exactness fact "pullbacks
preserve epics" that Freyd uses ambiently in the §1.974 proof but which this repo does NOT
derive from `Topos` — cf. the faithful sorries in §1.94 `topos_is_regular`).  Proved
constructively (`#print axioms` empty).

Helpers added (all in `[HasTerminal][HasExponentials]`, no choice):
- `cover_of_split_epi {f s} (hsf : s≫f = id) : Cover f` — a mono with a section through the
  factor is iso (mono-cancel), so any `f` with a right inverse is a cover.
- `expName g := curry (fst≫g) : 1 → A^C` — the *name* (point) of a map `g:C→A`.
- `expApply p := ⟨id_C, term≫p⟩ ≫ eval : C → A` — uncurry a point.
- `expApply_expName : apply ⌜g⌝ = g`, `expApply_postMap : apply(p≫f^C) = apply p ≫ f`,
  `expName_postMap : ⌜g⌝≫f^C = ⌜g≫f⌝` (curry/eval calculus).

**Forward** (∀ proj ⇒ IAC ∧ 1 proj): `1` projective is the `C:=1` instance.  IAC: a cover
`f:A→C` splits (`C` projective), `s≫f=id`; exponential functoriality
(`expCovMap_comp`/`expCovMap_id`) gives `s^A ≫ f^A = (s≫f)^A = id`, so `f^A = expPostMap A B C f`
is a split epi, hence a cover by `cover_of_split_epi`.

**Backward** (IAC ∧ 1 proj ⇒ ∀ proj): given a cover `f:A→C`, `q := f^C : A^C → C^C` is a cover
(IAC).  Pull `q` back along the name `⌜id_C⌝ : 1 → C^C`; the projection `π₂ : P → 1` is a cover
(`cover_pullback`), and `1` projective splits it: `r≫π₂ = id`.  Then `p := r≫π₁ : 1 → A^C`
lifts the name (`p≫q = ⌜id_C⌝`, via the pullback square `π₁≫q = π₂≫⌜id_C⌝`).  The uncurry
`s := apply p : C → A` is the section: `s≫f = apply(p≫f^C) = apply(p≫q) = apply ⌜id_C⌝ = id_C`.

### `nno_is_coproduct` (§1.985(1)) — `[0,s] : 1+N → N` is iso

Let `c = [0,s] : 1+N → N`.  Build the inverse by NNO-iterate:
`f := [0≫inr, s≫inr] : 1+N → 1+N` and `d := iterate inl f : N → 1+N`.

Three case-uniqueness facts:
- `inl≫c = 0`, `inr≫c = s` (`case_inl`/`case_inr`).
- `f ≫ c = c ≫ s`: both legs equal `case (0≫s) (s≫s)` (apply `case_uniq` to each side).

**`d ≫ c = id_N`** by NNO-uniqueness (`id_N = iterate 0 s`):
`0≫d≫c = inl≫c = 0` ✓; `s≫d≫c = (d≫f)≫c = d≫(c≫s) = (d≫c)≫s` ✓.

**`c ≫ d = id_{1+N}`** by `case_uniq` (`id = case inl inr`):
- inl-leg: `inl≫c≫d = 0≫d = inl` ✓ (`iterate_zero`).
- inr-leg: `inr≫c≫d = s≫d = d≫f`, and `d≫f = inr` by a SECOND NNO-uniqueness —
  both `d≫f` and `inr` equal `iterate (0≫inr) f` (for `inr`: `s≫inr = inr≫f` is `case_inr`;
  for `d≫f`: `s≫d≫f = (d≫f)≫f`).  No circular "[0,s] monic" argument is needed.

### `nno_peano_property` (§1.987) — the NNO has the Peano property

Take a subobject `B ↣ N` allowing `0` (point `e:1→B.dom`, `e≫arr=0`) and stable under
`s` (`tB:B.dom→B.dom`, `tB≫arr = arr≫s`).  Then `(B.dom, e, tB)` is an N-algebra and
`sec := iterate e tB : N → B.dom` is a **section** of `arr`:
`sec≫arr = iterate 0 s = id_N` by NNO-uniqueness (`0≫sec≫arr = e≫arr = 0`;
`s≫sec≫arr = (sec≫tB)≫arr = sec≫arr≫s`).  Since `arr` is monic, a section is a
two-sided inverse: `arr≫sec = id_{B.dom}` by mono-cancelling `arr` from
`(arr≫sec)≫arr = arr≫(sec≫arr) = arr = id≫arr`.  Hence `IsIso arr`, i.e. `B` is entire.

### `nno_is_free_one_action` (§1.98(12))

**Construction.** The NNO (N, 0, s) is a free 1-action with:
- `obj = N`, `unit = zero`, `act = snd ≫ succ : 1 × N → N`
- `recA α := hN.iterate α.unit (pair(term α.obj)(Cat.id α.obj) ≫ α.act)` = NNO iterate
  with step `f_rec α := pair(term,id) ≫ α.act : α.obj → α.obj` ("apply α.act to (*, x)").

**Key identity** (helper `prodMap_factorN`):
`prodMap one N B h = (snd ≫ h) ≫ pair(term B)(Cat.id B)`
proved by `pair_uniq`: fst component = `term` (by `term_uniq`), snd component = `snd ≫ h` ✓.

**recA_unit**: direct from `hN.iterate_zero`.

**recA_act**: `prodMap one N B iter ≫ α.act`
= `((snd ≫ iter) ≫ pair(term,id)) ≫ α.act` [prodMap_factorN]
= `snd ≫ iter ≫ (pair(term,id) ≫ α.act)` [Cat.assoc × 3, congr 1]
= `snd ≫ succ ≫ iter` [← iterate_succ with explicit args]
= `(snd ≫ succ) ≫ iter` [Cat.assoc].

**recA_uniq**: from hypothesis `hms : prodMap one N α.obj m ≫ α.act = (snd ≫ succ) ≫ m`,
applying `prodMap_factorN` and reassociating gives
`snd ≫ m ≫ f_rec α = snd ≫ succ ≫ m` (as maps `1 × N → α.obj`).
Cancel `snd` from the left using its section `prodOneLeftInv N` (which satisfies
`prodOneLeftInv N ≫ snd = id_N`, i.e., `snd` has a section, so is left-cancellable).
Conclude `m ≫ f_rec α = succ ≫ m`, then `hN.iterate_unique` closes the goal.

### `nno_terminal_is_coequalizer` (§1.985)

Given `f : N → X` with `succ ≫ f = f`, take `g = zero ≫ f`.
Uniqueness: `zero ≫ term N = id₁` by terminal uniqueness.
Existence: both `f` and `term N ≫ g` satisfy NNO equations for `(g, id_X)`,
so NNO uniqueness gives `f = term N ≫ g`.
