## Proofs discharged this session (§1.981 + §1.983)

All of §1.981 (NNO iterate for pairs) and §1.983 (primitive recursion) are now
proved from the NNO universal property + the exponential adjunction — pure
universal-property algebra, no deep topos infrastructure. `iteratePair_unique`
uses **no axioms**; the `primRec` family uses only `propext`.

- **iteratePair a b** = `(A × iterate a_hat b_hat) ≫ eval`, with
  `a_hat = curry(p₁≫a) : 1→B^A` and `b_hat = expPostMap A B B b = curry(eval≫b)`.
- **iteratePair_zero**: `⟨1,0⟩≫iteratePair = a`. Absorb the pair into the product
  functor (`pair_prodMap`), rewrite `0≫iter = a_hat` (NNO `iterate_zero`), then
  `(A×curry(p₁≫a))≫eval = p₁≫a` (`curry_eval_eq`); `⟨1,term⟩≫p₁ = 1`.
- **iteratePair_succ**: `(A×s)≫iteratePair = iteratePair≫b`. Collapse the two
  `prodMap`s on N (`prodMap_comp`), use NNO `iterate_succ` (`s≫iter = iter≫b_hat`),
  then `(A×b_hat)≫eval = eval≫b` since `b_hat = curry(eval≫b)`.
- **iteratePair_unique**: transpose `h` to `curry h : N→B^A`; show
  `curry h = iterate a_hat b_hat` by NNO `iterate_unique` (helper
  `prodMap_eval_inj` = injectivity of `g↦(A×g)≫eval`), then uncurry.
- **primRec g h** = `iteratePair ⟨⟨1,0⟩,g⟩ ⟨⟨p₁,p₂s⟩,h⟩ ≫ p₃` (the book's `k≫p₃`).
  `primRec_fst` proves the carrier component `k≫p₁ = 1_{A×N}` (so `k = ⟨p₁,p₂,f⟩`),
  again by §1.981 uniqueness. `_zero`/`_succ`/`_unique` then follow from the
  iteratePair equations exactly as Freyd's §1.983 argument.

## Proof discharged earlier

take g = zero≫f. Uniqueness: zero∘(term N) = id₁ by terminal uniqueness.
