compatibility through the intersection relation), hence a **Map** (`tabulated_is_map_iff_left_iso`);

unlock). Uniqueness is joint-epi: `case x y` is a cover (`union_case_cover`), hence epic.

1. `A₁ ∪ A₂ = f#(inl∪inr) = f#(⊤) = ⊤_A`  via `PreLogos.invImage_preserves_union` (S1_60 class field)
   + `inl_union_inr_entire` + `InverseImage f ⊤ ≅ ⊤`.
2. `A₁ ∩ A₂ ≤ f#(inl∩inr) ≤ f#(⊥) ≅ ⊥`  via `PreLogos.invImage_preserves_bottom` +
   `inl_inter_inr_le_bottom` — so the intersection apex is the bottom subobject, whose domain is
   **INITIAL** (`minimal_subobject_of_one_is_coterminator`, S1_61).
3. `pasting_lemma A₁ A₂` makes `A` (= `A₁∪A₂`) the pushout of the intersection legs;
   `pushout_over_initial_is_coproduct` (**proven this session, axiom-free**) then identifies it with
   `coprod A₁.dom A₂.dom`.
The only missing pieces are the steps-1/2 `InverseImage`-preservation arithmetic (entire-pullback iso
