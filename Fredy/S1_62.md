## Remaining sorries and their exact blockers

Four of the five remaining sorries bottom out at the **same missing fact**: in a pre-logos,
**composition distributes over union of relations** (Freyd §1.616, `compose_union_right` in
`Fredy/S1_60.lean`, which is itself a `sorry`), equivalently the **disjointness / pasting property
of coproduct inclusions**. The fifth sorry (`hh : Cover (case (y ≫ inl) inr)`) was **DISCHARGED** in
this session via an image-theoretic argument using `union_via_coproduct_image` (S1_61) and
`image_cover_comp`.

### `pasting_lemma` (§1.62) — BLOCKED on §1.616 `compose_union_right`

Freyd's proof (book p.101) sets `R = x°f ∪ y°g`, shows `1 ⊆ RR°` and `R°R ⊆ 1` so `R` is a map
(§1.564 simple+entire ⟹ map), then computes
`xR = x(x°f ∪ y°g) = xx°f ∪ xy°g = f`.
That last step **requires composition to distribute over `∪ᵣ`** — exactly the open direction
`Fredy/S1_60.lean:267 compose_union_right` (sorry) and the lattice-distributivity
`rel_inter_union_le` (sorry). The reverse inclusions (`compose_union_right_le`) are proven, but the
map-identity needs the forward (sorry) direction. There is no shortcut via
`pullback_of_covers_is_pushout` (S1_56) because the intersection legs `π₁,π₂` are not covers in
general. The relation/`graph`/`°`/`⊚`/`∪ᵣ` calculus all exists (S1_56/S1_60), so once
`compose_union_right` lands the book proof transcribes directly.

### `decompose_via_coproduct` (§1.624) — BLOCKED on `pasting_lemma`

The `Isomorphic A (coprod A₁ A₂)` claim is §1.621, an immediate corollary of the pasting lemma
(`A₁ = f#(inl)`, `A₂ = f#(inr)`, intersection `= 0`, union `= A`, so the pushout-of-intersection is
a coproduct). Needs `pasting_lemma`.

### `complemented_of_projective_is_projective` (§1.631) — 1 hole, BLOCKED on coproduct disjointness

The proof skeleton is complete. **DISCHARGED**: `hh : Cover (case (y ≫ inl) inr)` — proved via
`union_via_coproduct_image` (§1.615, S1_61): the image of `case (y ≫ inl) inr` is
`union(image(y ≫ inl), image inr)`, which is le-isomorphic to `union(image inl, image inr)` (using
`image_cover_comp` on `hy`) which equals the image of `case(inl, inr) = id` (entire). Mutual le +
monicity shows all these unions are entire, transferring to `image h` entire, i.e. `Cover h`. No
sorry required; uses only `S1_61.union_via_coproduct_image`, `S1_56.image_cover_comp`,
`S1_51.image_comparison_iso`, `S1_41.isIso_comp`, `S1_56.iso_cover`.

One remaining hole:
- `∃ r : P → B, r ≫ y = id` — needs `σ : P → B+P'` to factor through `inl_B : B → B+P'`, i.e. the
  `B`-summand is a *disjoint* (complemented) subobject of `B+P'`. This requires §1.621 coproduct
  disjointness (pasting lemma); BLOCKED.
