### `pasting_lemma` (§1.62) — DONE (this session)

Proven exactly per Freyd p.101. Carries `[HasBinaryCoproducts 𝒞]` in addition to `[PreLogos 𝒞]`
(a *positive* pre-logos, §1.623): the relational-union/distributivity infrastructure of §1.616
(`relUnion ∪ᵣ`, `compose_union_right`) is, in this repo, only built under `[HasBinaryCoproducts 𝒞]`,
and every downstream consumer (§1.624, §1.631) already works in a positive pre-logos.

Construction: the pushout cocone is `pt := (HasSubobjectUnions.union A₁ A₂).dom = U.dom`, with
`ι₁ = x`, `ι₂ = y` the two union inclusions (`x ≫ U.arr = A₁.arr`, `y ≫ U.arr = A₂.arr`). For an
arbitrary cocone `(Q, f, g)`, the descent morphism is extracted from the relation
`R = (graph x)°⊚graph f ∪ᵣ (graph y)°⊚graph g`, shown **Entire** (`1_U ≤ R⊚R°`, via the joint cover
`case x y`) and **Simple** (`R°⊚R ≤ 1_Q`, via four atomic terms — the one cross-term uses the cocone
compatibility through the intersection relation), hence a **Map** (`tabulated_is_map_iff_left_iso`);
`map_to_graph` yields `q` with `graph q ≃ R`. The factorizations `x ≫ q = f`, `y ≫ q = g` follow
purely from `graph x` being entire + `graph_faithful` (NO compatibility needed for fac — that was the
unlock). Uniqueness is joint-epi: `case x y` is a cover (`union_case_cover`), hence epic.

Uses the proven `compose_union_right` (S1_60), which itself transitively carries the isolated
extensivity `sorryAx` in `union_compose_descent` (S1_60) — so `pasting_lemma`'s axiom set is
`[propext, Classical.choice, sorryAx]`. No sorry of its own.

New helpers added to S1_62 (all real, sorry-free of their own): `map_to_graph`,
`cover_of_image_factor`, `union_is_image`, `union_case_cover`, `pairxx_factor`, `jcc_le`,

### `decompose_via_coproduct` (§1.624) — DONE (this session)

Builds `A₁ := f#(inlSub inl_mono)`, `A₂ := f#(inrSub inr_mono)`, legs `f₁,f₂` (second pullback
projections). The `Isomorphic A (coprod A₁.dom A₂.dom)` is now proven:

1. **Union entire** (`A₁ ∪ A₂ = ⊤_A`): `entire A ≤ f#(entire B)` (new helper `entire_le_invImage_entire`,
   the pullback lift of `⟨A, id_A, f⟩`) `≤ f#(Inl ∪ Inr)` (`invImage_mono_local` + `inl_union_inr_entire`)
   `= f#Inl ∪ f#Inr` (`PreLogos.invImage_preserves_union .1`). Then `entire A ≤ union ⟹ union entire`
   via new helper `entire_of_entire_le` (monic + section ⟹ iso).
2. **Intersection apex INITIAL**: build a cone `pbI.pt → (Inl ∩ Inr).dom` over `(inl,inr)` from the
   intersection legs (the central square chases `pbL/pbR.cone.w` through `f`); `inl_inter_inr_le_bottom`
   gives `(Inl∩Inr).dom → (⊥_B).dom`, and `(⊥_B).dom ≅ 0` (`bottom_dom_iso`) yields a map `pbI.pt → 0`,
   iso by `any_map_to_zero_is_iso` (S1_61) — so `pbI.pt ≅ 0`, hence any two maps out of it agree

3. `pasting_lemma A₁ A₂` makes `A` (=`A₁∪A₂`) the pushout of the intersection legs;

### `complemented_of_projective_is_projective` (§1.631) — DONE (this session, axiom-free)

Strengthened to `[DisjointBinaryCoproduct 𝒞]` (load-bearing: the proof consumes coproduct
disjointness via `coprod_inl_inr_disjoint_elt`). `omit [PreLogos 𝒞]` for the same diamond reason.
Removed the **extraneous** `(x : A ⟶ P) (hx : Cover x)` hypotheses (the book's §1.631 has no such
premise; the section is produced from the arbitrary cover `y` alone — keeping them would be an
unused-hypothesis defect).

The final hole `∃ r : P ⟶ B, r ≫ y = id_P` is discharged **directly** (the §1.624 *existential* is
too lossy to extract the factoring, so the disjointness arithmetic is rerun on `σ`):
- `P₁ := σ#(inl_B)`, `P₂ := σ#(inr_{P'})`.
- **`P₂.dom` initial**: its legs give `q₂ ≫ inl_P = g₂ ≫ inr_{P'}` in `P+P'` (compose the pullback
  square with `h`, using `σ≫h=inl` and `inr≫h=inr`); `coprod_inl_inr_disjoint_elt` + `bottom≅0` +
  `any_map_to_zero_is_iso` ⟹ `P₂.dom ≅ 0`.
- `P₂ ≤ ⊥_P ≤ P₁` (initial domain ⟹ unique triangle), so `union P₁ P₂ = P₁`; combined with
  `entire P ≤ union P₁ P₂` (same step-1 arithmetic on `σ`) ⟹ **`P₁` entire**, i.e. `q₁ := P₁.arr` iso.
- `q₁ = g₁ ≫ y` (push `q₁≫σ = g₁≫inl_B` through `h`, `inl_P` monic), so `r := q₁⁻¹ ≫ g₁` satisfies
  `r ≫ y = q₁⁻¹ ≫ q₁ = id_P`.

`lean_verify`: axioms `[propext, Classical.choice]` — **no `sorryAx`** (fully honest).
