## Remaining sorries and their exact blockers

All five remaining sorries bottom out at the same missing fact: **coproduct disjointness**
(= the §1.62 pasting lemma), which itself waits on **§1.616 `compose_union_right`** in
`Fredy/S1_60.lean`.

### `pasting_lemma` (§1.62) — BLOCKED on §1.616 `compose_union_right`

Freyd's proof (book p.101) sets `R = x°f ∪ y°g`, shows `1 ⊆ RR°` and `R°R ⊆ 1` so `R` is a map
(§1.564 simple+entire ⟹ map), then computes `xR = x(x°f ∪ y°g) = xx°f ∪ xy°g = f`.
That last step **requires composition to distribute over `∪ᵣ`** — exactly the open direction
`Fredy/S1_60.lean` `compose_union_right` (sorry) and the lattice-distributivity
`rel_inter_union_le` (sorry). The reverse inclusions (`compose_union_right_le`) are proven, but the
map-identity needs the forward sorry direction. There is no shortcut via
`pullback_of_covers_is_pushout` (S1_56, itself sorry) because the intersection legs π₁,π₂ are not
covers in general. The relation/`graph`/`°`/`⊚`/`∪ᵣ` calculus all exists (S1_56/S1_60), so once
`compose_union_right` lands the book proof transcribes directly.

### `decompose_via_coproduct` (§1.624) — BLOCKED on `pasting_lemma`

The `Isomorphic A (coprod A₁ A₂)` claim is §1.621, an immediate corollary of the pasting lemma
(`A₁ = f#(inl)`, `A₂ = f#(inr)`, intersection `= 0`, union `= A`, so the pushout-of-intersection is
a coproduct). Needs `pasting_lemma`.

### `complemented_of_projective_is_projective` (§1.631) — 1 hole, BLOCKED on coproduct disjointness

The proof skeleton is complete. **DISCHARGED (previous session)**: `hh : Cover (case (y ≫ inl) inr)` —
proved via `union_via_coproduct_image` (§1.615, S1_61): the image of `case (y ≫ inl) inr` is
`union(image(y ≫ inl), image inr)`, which is le-isomorphic to `union(image inl, image inr)` (using
`image_cover_comp` on `hy`) which equals the image of `case(inl, inr) = id` (entire). Mutual le +
monicity shows all these unions are entire, transferring to `image h` entire, i.e. `Cover h`.

One remaining hole (line 445):
- **Precise reduction**: from `hσh : σ ≫ h = inl_P` (where `h = case(y ≫ inl_P, inr_{P'})`) we
  derive that `σ ≫ case(y, z) = id_P` for ANY `z : P' → P` (proof: `h ≫ case(id_P, z) = case(y, z)`
  by `case_uniq`; then `σ ≫ h ≫ case(id_P, z) = inl_P ≫ case(id_P, z) = id_P`). Equivalently,
  `r := σ ≫ case(id_B, w)` satisfies `r ≫ y = id_P` for any `w : P' → B`. The proof needs exactly
  one morphism `z : P' → P` (or `w : P' → B`), which is unavailable in the proof state.
- **Blocker**: coproduct disjointness (§1.62 pasting lemma) would give `t := inr_{P'} ≫ φ_inv ≫ s'
  = inr_{P'}` (i.e., the second component of the section stays in the P'-summand), which forces
  `σ` to factor through `inl_B : B → B+P'` and yields the desired `w : P' → B = ∅`.
  Concretely: from `t = inr_{P'}`, we get `φ_inv ≫ s' = case(σ, inr_{P'})` by UMP, and then
  `σ ≫ case(id_B, w_anything) ≫ y = σ ≫ case(y, w_anything ≫ y) = id_P` using the key equation.
  BLOCKED on `pasting_lemma` (§1.62) / `compose_union_right` (§1.616).
