### §1.981 `iteratePair` equations (this session, all axiom-free / choice-free)

Let `a_hat = curry(fst≫a) : 1→B^A`, `b_hat = expPostMap A B B b : B^A→B^A`,
`H = iterate a_hat b_hat : N→B^A`. `iteratePair a b = prodMap A N (B^A) H ≫ eval`.

- **iteratePair_zero**: collapse `pair(id,term≫zero) ≫ prodMap H` into
  `pair(id, term≫zero≫H)`; `zero≫H = a_hat` (`iterate_zero`); recognise
  `pair(id,term≫a_hat) = pair(id,term) ≫ prodMap a_hat`; `≫eval = fst≫a`
  (`curry_eval_eq`), then `fst_pair`/`id_comp`.
- **iteratePair_succ**: `prodMap succ ≫ prodMap H = prodMap (succ≫H)`
  (`prodMap_comp`); `succ≫H = H≫b_hat` (`iterate_succ`); then
  `prodMap b_hat ≫ eval = eval ≫ b` (`curry_eval_eq`, since `b_hat=curry(eval≫b)`).
- **iteratePair_unique**: transpose `h` to `curry h : N→B^A`. Key local lemma
  `curry_precomp : g ≫ curry f = curry(prodMap g ≫ f)`. Show `curry h` solves the
  NNO recursion for `(a_hat,b_hat)` (zero eq uses `h0`+term-uniqueness, succ eq
  uses `hs`), so `iterate_unique` gives `curry h = H`; then `curry_eval_eq`
  yields `iteratePair a b = prodMap H ≫ eval = h`.
