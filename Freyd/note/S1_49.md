## Notes on §1.496 (now discharged)

The earlier worry that these need a *skeletal* hypothesis (iso ⇒ object equality) was
mistaken: object equality comes for free from the τ1-uniqueness axiom `tau1_unique`, which
equates two τ-tables (hence their source objects) once a `TableIso` between them is exhibited.

- `subterminator_iso_is_id`: given iso `f : T → T'` (T subterminator), `T'` is also a
  subterminator (`subterminator_of_iso`).  Build the two τ-tables `P = (T; id_T, f)` and
  `Q = (T'; f⁻¹, id_T')` over feet `[T, T']` (both in τ by `subterminator_pair_mem`); `f` is a
  `TableIso P ≅ Q` whose column equations are exactly `f ≫ f⁻¹ = id_T` and `f ≫ id_T' = f`.
  `tau1_unique` forces `P = Q`, hence `T = T'`; then `f = id_T` because `T` is a subterminator
  (`subterminator_maps_eq`).
- `subterminator_iso_unique`: the first component of the above (`T' = T`).
- `terminator_eq_one` (the book's "in particular, there is a unique terminator"): a
  *terminator* — a subterminator `T` whose `term T : T → one` is an iso — equals `one`.
  The general statement "*every* subterminator is the terminator" was **dropped as FALSE**
  (the empty set is a subterminator in `Set`, itself a τ-category, but is not the terminator).
  The book only claims uniqueness of the terminator, captured here with the essential
  `IsIso (term T)` hypothesis; proof is `subterminator_iso_unique` against `subterminator_one`.
