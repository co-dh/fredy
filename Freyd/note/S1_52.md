## Notes

### §1.521 — MISSING

The functor category `𝒮^A` (presheaves/functors `A → Set`) requires:
- A `NatTrans` type (natural transformations between functors)
- A `Cat` instance on functor types (with vertical composition of NatTrans)
- Limits and colimits computed pointwise in `𝒮^A`

None of these exist in the codebase (`grep -rn NatTrans Freyd/` returns nothing).
Record as MISSING until that infrastructure is built.

### §1.524 — `Projective` already in S1_57

`Projective` is part of §1.524 conceptually, but it is defined in `Freyd/S1_57.lean`
alongside the Choice-Projective equivalence (`choice_iff_projective`).  Since S1_57
imports S1_52, adding a second `Freyd.Projective` here would create a duplicate
declaration.  The key property of 1 being projective in a capital category is recorded
here as `capital_one_projective`.

### §1.526 — Pts functor as representation

The full statement "Γ is a `RepOfPreReg`" requires Set = `Type v` to be a
CartesianCategory instance (with equalizers etc.).  That instance lives in S1_55
which imports S1_52, making a circular import.  The three preservation properties
are instead stated directly as type-theoretic theorems:
- `pts_terminal_unique`: Γ(1) is singleton
- `pts_prod_iso_left/right`: Γ(A×B) ↔ Γ(A)×Γ(B)
- `pts_covers_of_capital`: Γ preserves covers when 1 is projective
